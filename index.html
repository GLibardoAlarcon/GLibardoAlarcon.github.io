Dashboard para redes de Internet

En este proyecto se crea un dashboard que muestra cada una de las tecnologías que se usan para dar el servicio de internet, junto con la cantidad de clientes por localidad y demás información relevante. Donde el usuario puede ver la información de manera intuitiva y por medio de filtros le permite ver opciones de mejora y oportunidades en el mercado.
